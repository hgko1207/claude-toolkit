# Claude Cowork 자동화 완전 가이드 — 매일 아침 카톡으로 오는 AI 브리핑 만들기

> Claude Cowork + 크롬용 Claude + Pley MCP + 플러그인 + 스케줄러를 엮어서, 코딩 한 줄 없이 매일 자동으로 돌아가는 AI 에이전트를 만드는 방법입니다.

## 핵심 요약

- Claude Cowork는 **답변을 주는 AI**가 아니라 **나 대신 일하는 AI 에이전트**를 만드는 도구입니다
- **크롬용 Claude + Pley MCP**로 웹 데이터 수집, **스킬**로 파일 생성, **플러그인**으로 워크플로우 패키징, **스케줄러**로 자동 실행을 연결합니다
- 이 조합으로 "매일 아침 10시에 증시 뉴스를 수집 → 보고서 작성 → 카카오톡 전송"이 **완전 자동화**됩니다
- 코어 패턴은 **수집 → 분석 → 산출물 → 전달** 4단계이며, 증시가 아닌 어떤 주제에도 적용 가능합니다

## Part 1: Claude Cowork란?

기존 Claude와 Cowork는 **근본적으로 다른 도구**입니다.

| 구분 | 기존 Claude | Claude Cowork |
|------|-----------|--------------|
| 역할 | **답변**을 주는 도구 | **일**을 해주는 에이전트 |
| 파일 접근 | 대화창 안에서만 처리 | 내 컴퓨터 폴더 직접 읽고/쓰기 |
| 웹 접근 | 검색 결과만 조회 | 브라우저 직접 조작 |
| 외부 서비스 | 제한적 | Gmail, 노션 등 다양하게 연결 |

한 줄로 정리하면 **"알려주는 AI"에서 "해주는 AI"로의 전환**입니다.

## Part 2: 사전 준비 및 설치

### 필요한 것 2가지

| 항목 | 설명 |
|------|------|
| Claude Pro 이상 구독 | 무료 플랜에서는 Cowork 사용 불가 |
| Claude 데스크톱 앱 | 웹 브라우저에서는 사용 불가 (반드시 데스크톱 앱) |

### 데스크톱 앱 설치 경로

1. Claude 사이트 접속
2. 왼쪽 하단 **"앱 및 확장 프로그램 받기"** 클릭
3. 본인 OS(Windows/Mac)에 맞는 설치 프로그램 다운로드
4. 실행 후 로그인

### 2026년 4월 업데이트 주의사항

이전에는 상단에 있던 **채팅/코워크/코드** 탭이 **왼쪽 사이드바 상단**으로 이동했습니다. 위치를 못 찾으시면 왼쪽 바를 확인하시면 됩니다.

## Part 3: 작업 폴더 설정

Cowork를 시작할 때 가장 먼저 하는 작업은 **Claude가 작업할 폴더**를 지정하는 것입니다.

### 폴더 생성

```
바탕화면/
└── 에이전트/
    └── 클로드/
```

폴더 이름은 본인이 기억하기 쉽게 자유롭게 지정할 수 있습니다.

### Cowork에 폴더 연결

1. Claude 앱 → 왼쪽 사이드바 → **코워크** 클릭
2. 폴더 선택 영역 → **"다른 폴더 선택"** 클릭
3. 방금 만든 폴더 지정
4. 권한 요청이 나오면 **"항상 허용"** 선택

### 보안 걱정은 없나요?

> "컴퓨터가 여러 개의 방이 있는 구조라면, 내가 허락한 방만 열리는 열쇠를 Claude에게 쥐어주는 것"

Claude는 지정한 폴더 안에서만 파일을 읽고 쓸 수 있습니다. 다른 폴더에는 접근할 수 없습니다.

## Part 4: 크롬용 Claude로 웹 데이터 수집

### 크롬용 Claude란?

Claude가 **여러분의 크롬 브라우저를 직접 조작**하는 기능입니다. API 연동이 필요 없고, 실제로 브라우저를 열어서 사람이 보는 것처럼 페이지를 읽습니다.

> 사람이 브라우저로 할 수 있는 건 Claude도 할 수 있습니다.

### 설치 방법

1. 크롬에서 **"크롬 확장 프로그램"** 검색 → 스토어 접속
2. **"Claude"** 검색
3. 크롬에 추가하기 클릭
4. Claude 앱 → 프로필 → 설정 → **"브라우저 사용"** 토글 활성화
5. 왼쪽 사이드바 → **"크롬용 Claude"** 클릭

### 사용 예시

Cowork 대화창에 이렇게 입력합니다:

```
크롬용 Claude를 활용해서 한국 경제 사이트에 접속한 다음
한국 증시와 미국 증시에 대해 조사해줘.
```

그러면 실제 크롬 창이 열리고 Claude가 직접 사이트를 탐색하면서 데이터를 수집합니다.

## Part 5: Pley MCP — 한국 서비스 연결

### MCP란?

**Model Context Protocol**의 약자입니다. AI와 외부 서비스를 연결하는 **표준 규격**입니다.

> 핸드폰 충전선 비유로 이해하면 쉽습니다. 예전에는 아이폰, 삼성폰 충전선이 달랐지만, 지금은 **USB-C**로 통일됐습니다. AI 세계에서도 각 서비스마다 연결 규격이 달랐는데, Anthropic이 **MCP**로 통일한 것입니다.

### Pley MCP가 특별한 이유

Claude 기본 커넥터는 대부분 해외 서비스(Gmail, 노션, 슬랙 등) 위주입니다. **Pley MCP는 한국 서비스 전용**입니다:

- 카카오톡
- 네이버
- 사람인
- 그 외 한국 서비스 다수

### 설치 순서

#### Step 1: Pley MCP 사이트 가입

1. 크롬에서 **"Pley MCP"** 검색
2. 홈페이지 접속 → **카카오 로그인**
3. 원하는 MCP 선택 → **"도구함에 추가"**

#### Step 2: 이번 워크플로우에 필요한 MCP 2가지

| MCP | 용도 |
|-----|------|
| **카카오톡 나와의 채팅방** | AI가 내 카카오톡으로 메시지 전송 |
| **네이버 검색** | 네이버 뉴스/증권 데이터 수집 |

#### Step 3: Claude 앱에 연결

1. Claude 앱 → 왼쪽 상단 **커스터마이즈** 클릭
2. **커넥터** 선택 → **플러스(+) 버튼**
3. **"커넥터 둘러보기"** → **"Pley MCP"** 검색
4. 노란 아이콘 플러스 버튼 클릭
5. **"모두 동의 후 속행"** 클릭

이렇게 연결하면 Pley MCP를 통해 카카오톡과 네이버에 Claude가 접근할 수 있게 됩니다.

### 사용 예시

```
네이버 증권에서도 정보를 가져와줘.
```

그러면 네이버 검색 MCP를 통해 실시간 증권 데이터를 수집합니다.

## Part 6: 보고서 생성 및 카카오톡 전송

### 워드 파일로 보고서 만들기

한국경제와 네이버에서 수집한 데이터를 합쳐서 워드 파일로 저장합니다:

```
상세한 보고서를 docx 파일로 만들어서
지금 작업하고 있는 폴더에 넣어줘.
```

Claude가 **docs 스킬**을 사용해서 아래 내용이 들어간 워드 파일을 자동 생성합니다:

- 핵심 요약
- 상위 종목 등락률
- 거시 경제 지표
- 주요 이슈

### 카카오톡으로 요약본 전송

```
보고서 내용을 카카오톡으로 보내줘.
```

Pley MCP의 카카오톡 MCP를 통해 **"나와의 채팅방"**으로 요약본이 전송됩니다.

### 여기까지의 전체 흐름

```
1. 한국경제 뉴스 수집 (크롬용 Claude)
2. 네이버 증권 데이터 수집 (Pley MCP)
3. 분석 보고서 워드 파일 생성 (docs 스킬)
4. 요약본 카카오톡 전송 (Pley MCP)
```

이 4단계가 **하나의 명령**으로 끊김 없이 실행됩니다.

## Part 7: 플러그인으로 패키징

### 플러그인이 필요한 이유

매번 같은 명령을 반복하는 건 비효율적입니다. **플러그인**으로 묶으면 슬래시 명령 하나로 전체 워크플로우를 실행할 수 있습니다.

### 플러그인 구성 요소

> 회사로 치면, 스킬이 **업무 매뉴얼**이라면 플러그인은 **신입사원 온보딩 키트**입니다.

한 플러그인에 다음이 전부 들어갑니다:

- 스킬
- 커넥터 설정
- 명령어
- 서브 에이전트

### 패키징 방법

Claude에게 이렇게 요청합니다:

```
지금까지 한 작업을 플러그인으로 패키징해줘.
네이버 MCP 작업과 크롬용 Claude 작업은
서브 에이전트로 놓고 병렬 작업식으로 패키징해줘.
```

그러면 Claude가 자동으로 플러그인을 생성합니다.

### 플러그인 저장

1. **"플러그인 설치"** 클릭
2. 우측 상단 **"플러그인 저장"** 클릭
3. 왼쪽 사이드바 → **커스터마이즈** → 저장 확인

### 사용 방법

| 방법 | 설명 |
|------|------|
| GUI | Cowork → 플러스(+) → 플러그인에서 선택 |
| 슬래시 명령 | `/generate-report` 한 줄 |

### 플러그인 공유

플러그인 파일은 작업 폴더에 자동 저장됩니다. 지인에게 파일을 보내면 **동일한 워크플로우**를 사용할 수 있습니다.

**주의**: 받는 사람도 동일한 MCP 커넥터들을 미리 연결해두어야 합니다.

## Part 8: 스케줄러로 완전 자동화

### 매일 자동 실행 설정

1. 왼쪽 사이드바 → **스케줄** 클릭
2. **"뉴 테스크"** 클릭
3. 아래와 같이 입력:

| 항목 | 값 |
|------|-----|
| 이름 | 증시 분석 (자유롭게) |
| 설명 | 증시 분석 |
| 프롬프트 | `/generate-report 실행해줘` |
| 빈도 | 매일 오전 10시 |
| 모델 | Sonnet 4.6 |
| 폴더 | 증시 분석용 새 폴더 |

4. **저장** 클릭

이제 매일 아침 10시에 자동으로 보고서가 생성되고 카카오톡으로 전송됩니다.

### 중요한 주의사항

> **컴퓨터와 Claude 앱이 반드시 켜져 있어야 합니다.**

컴퓨터가 꺼져 있으면 스케줄러도 동작하지 않습니다. 본인의 생활 패턴에서 **컴퓨터가 꼭 켜져 있는 시간대**로 설정하는 것을 추천합니다.

## Part 9: Claude Dispatch

### Dispatch란?

**핸드폰으로 데스크톱 Claude 앱을 조종**하는 기능입니다. 핸드폰이 일종의 리모컨 역할을 합니다.

### 사용 방법

1. 핸드폰 Claude 앱 다운로드
2. 왼쪽 → **Dispatch** 메뉴
3. 명령 입력

그러면 모바일 명령이 PC Claude 앱으로 전달되어 실행됩니다.

### 현재 상태 (2026년 4월 기준)

최근 Claude 업데이트 과정에서 **지시사항이 컴퓨터로 제대로 넘어가지 않는 버그**가 종종 발생하고 있습니다. 안정화될 때까지는 주의해서 사용하시는 것이 좋습니다.

## 이 구조를 다른 주제에 적용하기

증시 브리핑은 하나의 예시일 뿐입니다. 동일한 패턴이 어디에나 적용됩니다.

| 활용 예시 | 수집 | 분석 | 산출물 | 전달 |
|----------|------|------|-------|------|
| 경쟁사 모니터링 | 경쟁사 뉴스 사이트 | 변화 요약 | 보고서 | 슬랙 |
| 고객 리뷰 분석 | 리뷰 사이트 | 감성 분석 | 정리 문서 | 노션 |
| 유튜브 트렌드 | 인기 영상 조회 | 주제 분류 | 트렌드 리포트 | 이메일 |
| 환율/원자재 체크 | 금융 사이트 | 변동률 계산 | 일일 보고서 | 카카오톡 |

**핵심 패턴**: 수집 → 분석 → 산출물 → 전달

이 4단계 구조를 이해하면 본인만의 자동화 워크플로우를 설계할 수 있습니다.

| 7. 자동화 | 매일 지정 시간 실행 | 스케줄러 |

Cowork의 진짜 가치는 **단일 작업**이 아니라 **워크플로우**에 있습니다. 도구 하나를 잘 쓰는 것이 아니라, 도구들을 연결해서 **시스템을 설계**하는 것입니다. 그리고 Cowork는 그 시스템을 **코딩 한 줄 없이** 만들 수 있게 해줍니다.

**"AI를 도구로 쓰는 단계에서 AI로 에이전트를 설계하는 단계로"** 넘어가는 전환점입니다.
