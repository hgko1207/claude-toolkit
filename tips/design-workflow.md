# AI Slop 극복 — Claude Code로 디자인 잘하는 법

> AI가 만든 티 나는 촌스러운 디자인(AI Slop)을 벗어나는 실전 워크플로우입니다.
> 디자인 스킬 + Gemini 영상 + 템플릿 조합으로 프로급 랜딩 페이지를 만듭니다.
>
> 출처: [99%가 모르는 클로드 코드로 디자인 가장 잘 하는 방법](https://www.youtube.com/watch?v=2sNQ0Nvngdc)

---

## AI Slop이란?

AI가 AI스럽게 만든 디자인을 말합니다.

**흔한 증상:**
- 보라색 그라데이션 남용
- 어색한 폰트 조합
- 뻔한 레이아웃 (히어로 + 3칼럼 카드)
- 스톡 이미지 느낌의 일러스트
- 전체적으로 "어디서 많이 본" 느낌

Claude Code는 기능적으로 뛰어나지만, 디자인만큼은 이런 AI Slop에서 벗어나기 어렵습니다. 이걸 해결하는 방법이 **디자인 스킬**입니다.

## 수파노바 디자인 스킬이란?

해외에서 만들어진 **taste-skill**(디자인 감각 스킬)을 한국 환경에 맞게 커스터마이징한 것입니다.

**기존 taste-skill 문제:**
- 영문 폰트 최적화만 되어 있음
- 한국어로 만들면 폰트가 어색함
- 아이콘이 해외 스타일

**수파노바 디자인 스킬 개선점:**
- **프리텐다드 폰트** 적용 (한국어에 최적화된 폰트)
- **Iconify** 아이콘 시스템 사용
- 한국어 랜딩 페이지에 특화된 디자인 지침
- 한국 사용자를 위한 현지화

## 워크플로우 전체 순서

### Step 1: 디자인 스킬 설치

GitHub에서 수파노바 디자인 스킬을 다운로드합니다:

```bash
# 프로젝트의 .claude/skills/ 폴더에 복사
git clone https://github.com/uxjoseph/supanova-design-skill .claude/skills/supanova
```

또는 SKILL.md 파일만 프로젝트에 복사해서 사용할 수 있습니다.

---

### Step 2: Claude Code로 초안 생성

Claude Code에 스킬 GitHub URL을 함께 전달하면서 웹사이트 생성을 요청합니다:

```
[수파노바 디자인 스킬 GitHub URL 붙여넣기]

이 스킬을 기반으로 인테리어 업체 소개를 위한 웹사이트를 하나 만들어 주세요.
```

**결과**: 프리텐다드 폰트가 적용된 깔끔한 초안이 나옵니다. taste-skill 기반이라 AI Slop 없이 세련된 디자인입니다.

---

### Step 3: Gemini로 히어로 영상 생성

#### 3-1. 레퍼런스 이미지 찾기

Pinterest에서 원하는 분위기의 이미지를 검색합니다:

```
Pinterest → "인테리어 디자인" 검색 → 마음에 드는 이미지 복사
```

#### 3-2. Gemini에서 영상 생성

1. [Gemini](https://gemini.google.com/) 접속
2. **동영상 만들기** 선택
3. 레퍼런스 이미지 붙여넣기
4. 프롬프트 입력:

```
이 이미지를 바탕으로 천천히 화면이 돌아가는 동영상을 하나 16:9 비율로 생성해 줘
```

5. **프로 모델** 선택 (품질 차이가 큽니다)
6. 생성된 영상 다운로드

#### 3-3. 다양한 스타일 시도

**인테리어 스타일:**
```
이 이미지를 바탕으로 그라운드 뷰로 천천히 재생되는
랜딩 페이지 히어로에 넣을 16:9 영상을 하나 만들어 줘
```

**추상 도형 스타일:**
```
이 요소가 중심점을 기준으로 천천히 돌아가는 16:9 영상을 하나 만들어 줘
```

Pinterest에서 그라데이션이 들어간 추상적인 도형 이미지를 찾아서 전달하면 히어로 섹션을 풍부하게 꾸밀 수 있습니다.

---

### Step 4: 영상을 웹사이트에 결합

생성된 영상을 프로젝트 폴더에 저장한 후 Claude Code에 지시합니다:

```
[영상 파일을 프로젝트 폴더에 드래그 앤 드롭]

이 영상을 히어로 섹션에 넣어 주고 스크롤 애니메이션이 적용되게 해 줘.
스크롤 애니메이션은 WebP 파일로 변환해서 만들어 줘.
부드러운 애니메이션이 될 수 있도록 꼭 신경 써서 해 줘.

나머지 이미지들도 인테리어 웹사이트에 걸맞도록 교체해 줘.
```

**결과**: 스크롤 방향에 따라 영상이 부드럽게 전환되는 히어로 섹션이 완성됩니다.

---

### Step 5: 템플릿으로 디자인 레벨업 (선택)

수파노바 템플릿(200+ 종)에서 마음에 드는 디자인을 골라 코드를 복사합니다:

1. [수파노바](https://www.supanova.dev/) → 템플릿 둘러보기
2. 마음에 드는 템플릿 선택 → 미리보기
3. **내보내기** → **HTML 코드 복사**
4. Claude Code에 붙여넣기:

```
[HTML 코드 붙여넣기]
[영상 파일 첨부]

이 영상과 코드를 결합한 다크모드 형태의 웹사이트를 하나 만들어 줘
```

**결과**: 템플릿의 세련된 레이아웃 + Gemini 영상이 결합된 프로급 랜딩 페이지입니다.

---

### Step 6: Netlify Drop으로 무료 배포

#### 6-1. zip 파일 생성

```
이 HTML 파일과 연관된 파일을 함께 zip 파일로 압축해 줘.
Netlify Drop용이야.
```

#### 6-2. 업로드 및 배포

1. [Netlify Drop](https://app.netlify.com/drop) 접속
2. 생성된 zip 파일을 드래그 앤 드롭
3. 업로드 완료 → **고유 URL 자동 생성**
4. 전 세계 어디서든 접속 가능한 라이브 웹사이트 완성

서버 설정, 도메인 구매 등 복잡한 과정 없이 **zip 파일 하나로 배포가 끝납니다**.

---

## 전체 워크플로우 요약

```
1. 수파노바 디자인 스킬 설치
   → Claude Code의 디자인 감각 업그레이드

2. Claude Code로 웹사이트 초안 생성
   → AI Slop 없는 깔끔한 기본 구조

3. Pinterest에서 레퍼런스 이미지 찾기
   → 원하는 분위기/스타일 설정

4. Gemini로 히어로 영상 생성 (프로 모델)
   → 시네마틱한 배경 영상

5. Claude Code에서 영상 + 웹사이트 결합
   → 스크롤 애니메이션 적용

6. (선택) 수파노바 템플릿 코드 결합
   → 프로급 레이아웃

7. zip 압축 → Netlify Drop 업로드
   → 라이브 배포 완료
```

## 꿀팁

### Claude Code 단축어 설정

매번 `claude`를 입력하기 귀찮으면 단축어를 설정하세요:

```bash
# .bashrc 또는 .zshrc에 추가
alias cc="claude"
```

이후 `cc`만 입력하면 Claude Code가 실행됩니다.

### Gemini 프롬프트 팁

- **"천천히"**를 꼭 넣으세요 — 빠른 영상은 히어로 섹션에 어울리지 않습니다
- **16:9 비율** 명시 — 웹사이트 배경에 최적입니다
- **레퍼런스 이미지**를 항상 함께 제공 — 텍스트만으로는 원하는 결과가 나오기 어렵습니다
- **프로 모델** 선택 — 무료 모델과 품질 차이가 상당합니다

### WebP 변환 주의

스크롤 애니메이션을 위해 영상을 WebP 프레임으로 변환할 때:
- Claude Code에게 **"부드러운 애니메이션이 될 수 있도록 신경 써서"** 명시하세요
- 프레임 수가 많을수록 부드럽지만 파일 크기도 커집니다
- 120프레임 정도가 적당합니다

### 이미지 교체

Claude Code가 기본으로 넣는 이미지는 Unsplash 무료 이미지입니다. 마음에 안 들면:
```
이미지들을 [원하는 스타일]에 맞게 교체해 줘
```
