## 컨벤션

- 타입: `any`, `unknown` 사용 금지
- 커밋: 한글로 작성
- 브랜치: main
- 코드 스타일: (프로젝트에 맞게 작성)

## 주요 명령어

```bash
npm run dev        # 개발 서버
npm run build      # 프로덕션 빌드
npx tsc --noEmit   # 타입체크
npm test           # 테스트
```

## 폴더 구조

```
src/
├── app/           # 페이지/라우트
├── components/    # 재사용 컴포넌트
├── lib/           # 유틸리티, 타입
├── hooks/         # 커스텀 훅
└── data/          # 데이터/상수
```

## 에이전트 사용법

| 명령 | 용도 |
|------|------|
| `@planner 기능 추가해줘` | plan.md에 플랜 작성 |
| `@implementer 구현해줘` | plan.md 기반 코드 구현 |
| `@reviewer 리뷰해줘` | 코드 리뷰 |
| `@deployer 배포해줘` | 빌드 → 커밋 → 배포 |

## 스킬 사용법
